# What's in the folders above

# Have been uploaded 
1. ## Chatbot ELIZA
flowchart TD
    A[User Input Text] --> B[Clean & Tokenize Input String]
    B --> C{Match Keywords/Patterns?}
    C -->|Yes| D[Select Highest Priority Rule]
    C -->|No| E[Apply Fallback Generic Prompt]
    D --> F[Perform Deictic Shift <br> e.g., swap 'my' to 'your']
    F --> G[Assemble Response Template]
    E --> H[Output Response to User]
    G --> H
    H --> A
